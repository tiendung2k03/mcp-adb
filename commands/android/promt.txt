
Tạo một gói danh sách lệnh extension hiện tại với **tất cả các lệnh ADB phổ biến**, mỗi lệnh là 1 file `.toml`.  

Yêu cầu chi tiết:

1. **Tên file và tham số**: Danh sách file theo nhóm, nếu lệnh cần nhập tham số, ghi rõ trong ngoặc. Ví dụ: `connect.toml (nhập IP:PORT)`.

2. **Nội dung file `.toml`**: 
```toml
prompt = """
<Mô tả hành động, ví dụ lệnh adb, yêu cầu nhập tham số nếu cần>
"""
Nếu lệnh cần param, ghi rõ: “nhập <tên param>”.
Nếu lệnh chạy trực tiếp, ghi ví dụ thực thi: adb <lệnh>.

Yêu cầu Gemini CLI:
Tạo từng file .toml theo danh sách.
Mỗi file có nội dung prompt đầy đủ mô tả hành động.
Nếu cần nhập param, ghi hướng dẫn nhập param.
Nếu lệnh trực tiếp, ghi ví dụ thực thi adb <lệnh>.
